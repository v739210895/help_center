# 答题数据在线查看

在统计页可在线查看答卷详情，以快速了解问卷各题的回收数据及回收情况。

{% hint style="info" %}
用户未回答的题目答案处显示为空。
{% endhint %}

![&#x5728;&#x7EBF;&#x67E5;&#x770B;&#x7B54;&#x9898;&#x6570;&#x636E;](../../.gitbook/assets/image%20%28583%29.png)

矩阵单选题、矩阵多选题和矩阵量表题可点击弹窗查看每个问题的答题详情。

![&#x77E9;&#x9635;&#x5355;&#x9009;&#x9898;&#x67E5;&#x770B;&#x7B54;&#x9898;&#x8BE6;&#x60C5;](../../.gitbook/assets/image%20%28576%29.png)

支持按指定题目的答案或用户信息进行数据筛选，根据指定条件查看部分答卷。

![&#x6570;&#x636E;&#x7B5B;&#x9009;](../../.gitbook/assets/image%20%28581%29.png)
