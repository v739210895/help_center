# 统计图表

在统计图表页支持在线查看各题答案的统计结果、统计结果导出（含小计和百分比）。

{% hint style="info" %}
左侧导航大纲支持点击定位到指定题目。
{% endhint %}

![&#x7EDF;&#x8BA1;&#x56FE;&#x8868;](../../.gitbook/assets/image%20%2891%29.png)
