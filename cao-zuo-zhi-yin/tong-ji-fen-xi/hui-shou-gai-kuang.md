# 回收概况

### 整体概况

回收概况展示问卷回收的整体情况。

<figure><img src="../../.gitbook/assets/image (1184).png" alt=""><figcaption></figcaption></figure>

| 指标     | 说明                                  |
| ------ | ----------------------------------- |
| 回收量    | 实时回收的有效答卷数量                         |
| 浏览量    | 实时点击进入问卷投放链接的数量                     |
| 回收率    | 有效回答问卷的数量在浏览量中的占比，回收率=回收量/浏览量\*100% |
| 平均完成时间 | 回收答卷中所有答题者的平均答题时长，计算公式=所有用户总用时/回收量  |
| 用时中位数  | 回收答卷中所有答题者回答问卷的答题时长中位数              |
| 答题行为分析 | 统计单题中断人数与平均答题时间（必须在回收设置页面）          |

### 页面PV

点击浏览量指标的ICON，可以显示浏览量详情，系统会埋点统计各核心页面的PV。

<figure><img src="../../.gitbook/assets/image (1185).png" alt=""><figcaption></figcaption></figure>

| 指标     | 说明                                 |
| ------ | ---------------------------------- |
| 总浏览量   | 点击问卷链接的统计量                         |
| 登录成功人数 | 访问QQ登录页/微信授权登录/游戏内登录成功的数量          |
| 欢迎页PV  | 访问欢迎语的PV。海外验证流程可跳过欢迎页，此时，不会统计欢迎页PV |
| 答题页PV  | 访问答题页的PV，多个题目分页都只会统计1次             |

### 导出按日统计的页面PV

点击浏览量详情弹窗中的导出按钮，可以导出按天统计的指标数据。
