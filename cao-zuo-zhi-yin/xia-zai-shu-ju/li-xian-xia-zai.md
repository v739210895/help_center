# 离线下载

为提高文件导出效率，问卷系统采用文件异步下载方式，即先导出文件再下载到本地。在导出答题编码数据、导出主观题原始数据时，导出完成后需在顶部工具栏中的“离线下载”功能中下载对应文件。

{% hint style="info" %}
导出后的文件存储在云端，支持用户自行删除。
{% endhint %}
