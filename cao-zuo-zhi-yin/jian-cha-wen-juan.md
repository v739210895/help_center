# 检查问卷

问卷编辑后，可通过开启检查模式检查逻辑设置，以保证问卷内容的准确性。

![&#x68C0;&#x67E5;&#x6A21;&#x5F0F;](../.gitbook/assets/image%20%28575%29.png)

## 逻辑检查

开启检查模式后，问卷编辑页中已设置简单逻辑的单选题、多选题选项后会显示选中后显示的题目题号。

{% hint style="info" %}
仅支持简单逻辑检查，已启用组合逻辑的问卷将无法进行逻辑检查。
{% endhint %}
