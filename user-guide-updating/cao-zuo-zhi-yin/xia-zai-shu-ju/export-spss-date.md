# Export spss date

Select a survey from the "Home" list, click "Export" in the action bar, and choose "spss date" to export the survey file in sav format.

<figure><img src="../../../.gitbook/assets/image (1194).png" alt=""><figcaption></figcaption></figure>

The exported files can be opened directly using SPSS software for subsequent data analysis operations.
